# ShipMyMark Source Notes

Public source notes for [ShipMyMark](https://shipmymark.com/), an AI-assisted
logo and brand-mark tool.

This repository is a lightweight public reference for what the site is, which
public pages are intended for users, and how the project should describe itself
without overclaiming.

## Useful Pages

- [ShipMyMark homepage](https://shipmymark.com/)
- [ShipMyMark editor](https://shipmymark.com/editor)
- [Logo maker entry](https://shipmymark.com/)

## Product Boundary

ShipMyMark should be described as a practical logo and mark-generation tool. It
should not claim trademark clearance, legal brand ownership, guaranteed
uniqueness, or agency-level identity strategy unless those checks are actually
provided.

## Public Notes

- The public site is live at `shipmymark.com`.
- Generated assets should be reviewed by the user before commercial use.
- Legal trademark checks are outside a normal logo-generation workflow.

See [`data/public-pages.json`](data/public-pages.json) for the reference links.
